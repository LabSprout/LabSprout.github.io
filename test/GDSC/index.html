JR渋谷駅からGoogleオフィスの行き方

  1. 中央改札もしくは南改札を出る
  2. 東口渋谷スクランブルスクエア2階貫通路を経由して直結

銀座線からGoogleオフィスの行き方

  1. 下車後に一号車側に向かって進みます
  2. 改札を出ます
  3. 少し歩いて左に進みます
  4. 建物を出て左へ進み、SHIBUYA SCRAMBLE SQUAREが出てきます
  5. 前を左に進み、10mほど先の下りエスカレーターを降ります
  6. 降りた後に、右に進みます
  7. スクランブルスクエアの中を進み、50mほど先の突き当たりを左へ進みます(この時正面に渋谷ストリームビルが見えてきます)
  8. 数メートル先を右へ進み、ストリームビルに到着です

半蔵門線・田園都市線からGoogleオフィスの行き方

  1. 1号車1番ドアから降ります
  2. 宮益坂中央改札から出ます
  3. 改札から出た後、C2出口方面へ進みます
  4. 白い壁に突き当たったところを右に曲がります
  5. 右折してすぐに渋谷ストリームのアーバン・コアがあるので、ここからエスカレーターで上がります
  6. エスカレータを降りて右に回り込むと‥
  7. 渋谷ストリームの1Fエントランスに到着します

井の頭線からGoogleオフィスの行き方

  1. 進行方向一番前で降ります
  2. 改札を出ます
  3. 柱の右側を歩きます
  4. 100m位先を右に進みます
  5. 11段の階段を降ります
  6. 道なりに進みます
  7. 左へ進みます
  8. さらに通路を30秒歩きます
  9. 突き当りを左へ進みます
  10. 8段の階段があるので上がります
  11. さらに階段があるので上がります
  12. 20ｍ程先「中央改札」の案内が見えてきます
  13. 斜め右に進みます
  14. 「SHIBUYA SCRAMBLE SQUARE」入り口です。入り口手前を左に進みます
  15. 少し歩くと下りエスカレータ​​
  16. 下りて右へ進みます
  17. スクランブルスクエアの中を進みます
  18. 50m程先の突き当りを左へ進みます
  19. 正面には渋谷ストリームのビルが見えてきます
  20. 数メートル先を右へ進みます
    21. ‹

副都心線・東横線からGoogleオフィスの行き方

  1. 一号車一番目のドアから降ります
  2. エスカレータまたは階段で上がります
  3. ヒカリエ１改札案内を見つけたら右に曲がります
  4. 通路を６０メートルほど歩きます
  5. ミニエスカレーターを降ります
  6. 降りた先にある半球身体のオブジェの左側に行きます
  7. ヒカリエ１改札を出て右に進みます
  8. 正面に渋谷スクランブルスクエアが見えたら左に曲がります
  9. C1からC3出口のところに進みます
  10. そのまま道なりに進みエスカレータを上るとGoogleという看板があるのでその矢印の方向に進みます
